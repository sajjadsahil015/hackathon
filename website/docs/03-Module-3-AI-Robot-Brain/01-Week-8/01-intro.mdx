# Intro to Isaac Sim

## What is Isaac Sim?
Isaac Sim is built on **NVIDIA Omniverse**. It leverages RTX ray-tracing for physically accurate sensor simulation.

## Key Features
*   **Synthetic Data Generation (SDG)**: Train models on 1M+ generated images.
*   **Isaac Gym**: Reinforcement Learning on GPU.
*   **ROS 2 Bridge**: Seamless integration.

## Hardware Check
<div className="alert alert--danger">
  <strong>Requirement:</strong> NVIDIA RTX 2070 or higher.
</div>
