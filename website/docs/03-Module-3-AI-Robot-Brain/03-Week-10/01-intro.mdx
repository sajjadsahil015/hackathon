# Navigation 2 (Nav2)

Nav2 is the standard navigation framework for ROS 2.

## Core Components
1.  **Global Planner**: Finds the optimal path from A to B (Dijkstra/A*).
2.  **Local Controller**: Follows the path avoiding dynamic obstacles (DWB/MPPI).
3.  **Costmaps**: 2D grids marking safe and unsafe areas.
4.  **Recovery Behaviors**: What to do if stuck (Spin, Back up).

## Costmap Layers
*   **Static Layer**: Walls from the SLAM map.
*   **Obstacle Layer**: Real-time Lidar/Camera hits.
*   **Inflation Layer**: Padding to prevent collisions.
