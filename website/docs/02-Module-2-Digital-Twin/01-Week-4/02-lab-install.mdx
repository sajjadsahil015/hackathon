# Lab: Installing Gazebo

## Installation

<TerminalBlock commands={{
  linux: "sudo apt install ros-humble-gazebo-ros-pkgs",
  windows: "wsl install ros-humble-gazebo-ros-pkgs",
  mac: "brew install gazebo"
}} />

## Running the Demo

```bash
ros2 launch gazebo_ros gazebo.launch.py
```

You should see an empty 3D world.
