# Lab: Maze Run

## Setup
1. Download the "Turtlebot3 World" maze.
2. Spawn your robot URDF into the maze.

## Challenge
Drive the robot from Start to Finish using the keyboard.

```bash
ros2 run teleop_twist_keyboard teleop_twist_keyboard
```

Observe the Lidar data in RViz as you approach walls.
