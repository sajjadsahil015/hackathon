# URDF Basics

**URDF** is XML for robots.

## Anatomy of a Robot

```xml
<robot name="my_robot">
  <!-- The Main Body -->
  <link name="base_link">
    <visual>
      <geometry>
        <box size="0.6 0.4 0.2"/>
      </geometry>
    </visual>
  </link>
  
  <!-- A Wheel -->
  <link name="right_wheel">...</link>
  
  <!-- The Joint -->
  <joint name="base_to_wheel" type="continuous">
    <parent link="base_link"/>
    <child link="right_wheel"/>
  </joint>
</robot>
```
