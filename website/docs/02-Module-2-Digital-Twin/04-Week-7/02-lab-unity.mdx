# Lab: Unity-ROS Bridge

## Setup
1. Install **Unity Hub** and **Unity 2022.3 LTS**.
2. Import the **Unity Robotics Hub** package.

## Exercise: Pick & Place Visualization
1. Import a UR3e robot arm into Unity.
2. Create a ROS 2 node that publishes `JointTrajectory`.
3. Configure the Unity `RoboticsConnector` component.
4. Watch the arm move in Unity as you publish commands from the terminal!
