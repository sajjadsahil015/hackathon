# Actions & Parameters

## Actions

Actions are goal-oriented tasks.
*   **Example**: "Navigate to Waypoint A"
*   **Feedback**: "Distance remaining: 5m... 4m..."
*   **Result**: "Arrived at A".

## Parameters

Configuration values stored on the node.
*   `max_speed`: 2.0 m/s
*   `camera_iso`: 800
