# Research: Interactive Robotics Textbook

**Feature**: Interactive Robotics Textbook
**Date**: 2025-12-07

## Decisions

### 1. Navigation Hierarchy & Sidebar Logic
- **Decision**: Use Docusaurus "Autogenerated Sidebars" with directory-based ordering.
- **Rationale**:
  - `docs/01-Module-1/...` automatically sorts first.
  - Reduces manual maintenance of `sidebars.ts`.
  - Supports the deep nesting requirement (Module > Week > Topic) out of the box.
- **Alternatives Considered**:
  - *Manual Sidebar Definition*: Too brittle for a 13-week course that might expand.
  - *Flat Structure*: Violates "Structural Fidelity" principle.

### 2. Styling Strategy
- **Decision**: Hybrid approach - Infima (Base) + Tailwind CSS (Utility).
- **Rationale**:
  - Infima handles layout, dark mode, and accessibility defaults (Constitution requirement).
  - Tailwind allows rapid creation of "Card" components for the homepage and custom tables for Hardware Specs without writing heavy custom CSS.
- **Alternatives Considered**:
  - *Pure Infima*: Harder to create custom "Academic" visual flair.
  - *Pure Custom CSS*: Reinventing the wheel, prone to accessibility bugs.

### 3. Asset Management
- **Decision**: Standard `static/img` folder + Mermaid.js for diagrams.
- **Rationale**:
  - "Constitution Principle: Assets" prefers Mermaid.js.
  - Git LFS is complexity overkill for a text-heavy textbook unless we have massive binary datasets (which are excluded by "Frontend Focus").
- **Alternatives Considered**:
  - *External Hosting (S3)*: Violates "No Backend/Auth" simplicity; adds deployment complexity.

### 4. Context7 MCP Integration
- **Decision**: Build a custom Docusaurus plugin or script to fetch/sync content.
- **Rationale**:
  - FR-008 requires consuming content from Context7.
  - Docusaurus supports build-time plugins to fetch data and generate MDX.
  - Keeps the site static (fetches at build time, not runtime).

## Open Questions Resolved

- **Q**: How to handle "Quarter" grouping?
- **A**: Implicit. The root of `docs/` represents the Quarter. Modules are top-level folders.

- **Q**: Mobile sidebar behavior?
- **A**: Docusaurus default mobile navigation handles nesting via a collapsible menu drawer. This meets the "collapse gracefully" edge case.
